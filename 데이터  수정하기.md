## 데이터 수정 1단계
1. 수정 페이지를 만들고 수정 페이지를 컨트롤러에 요청한다.
2. 리파지터리에서 findByID(id) 후, DB에서 데이터를 찾아 다시 서버로 전송한다.
3. 가져온 데이터를 모델에 등록한다.
4. 뷰 페이지를 출력한다.

## 데이터 수정 2단계
1. 폼 데이터를 전달한다.
2. 컨트롤러에서 DTO를 리파지터리의 엔티티로 변환한다.
3. DB를 갱신(저장)한다.
4. 결과를 볼 수 있게 상세 페이지로 리다이렉트한다.
